Color Mixer

Dojo has a nice convenient utility for mixing two colors, including balancing towards one of the colors. This little demo shows off the color picker as well as balancing.

How to work with it:

  • Drag the slider left and right to adjust the balance of the mix.
  • Click on one of the top color blocks to display a color picker, which you can use to directly select a color.
  • Use either of the randomize buttons to set the color at random (duh)
